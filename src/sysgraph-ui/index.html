Search Syntax
term — fuzzy match (all fields)
field:value — match against specific field
properties.name:bash — dot notation for nested fields
field:"val:ue" — quotes for values with colons
"literal term" — quotes suppress all special meaning
AND / OR — logical operators (uppercase)
(A OR B) AND C — parentheses for grouping
Space between terms = implicit AND
